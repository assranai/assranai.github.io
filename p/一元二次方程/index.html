一元二次方程

Dec 28, 2025
阅读时长: 2 分钟

一元二次方程与二次函数完整笔记

📌 第一部分：一元二次方程

1.1 基本形式

一元二次方程是形如

$$ax^2 + bx + c = 0$$

（其中

$$a \neq 0$$

）的方程。

1.2 求根公式

$$ x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} $$

1.3 判别式 Δ

$$ \Delta = b^2 - 4ac $$
Δ 的值 根的性质 说明
Δ > 0 两个不相等的实数根 $$x_1 \neq x_2$$
Δ = 0 两个相等的实数根 $$x_1 = x_2 = -\frac{b}{2a}$$
Δ < 0 两个共轭复数根 实数范围内无解

1.4 韦达定理（根与系数关系）

$$ x_1 + x_2 = -\frac{b}{a}, \quad x_1 \cdot x_2 = \frac{c}{a} $$

扩展公式：

  • $$x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1x_2$$
  • $$|x_1 - x_2| = \frac{\sqrt{\Delta}}{|a|}$$

📈 第二部分：二次函数

2.1 二次函数基本形式

$$ y = ax^2 + bx + c \quad (a \neq 0) $$

2.2 与一元二次方程的关系

  • 二次函数的零点就是对应方程的根
  • 方程 $$ax^2 + bx + c = 0$$的解是函数 $$y = ax^2 + bx + c$$与 x 轴的交点横坐标

2.3 顶点坐标公式

二次函数的顶点坐标为：

$$ \left( -\frac{b}{2a}, \ \frac{4ac - b^2}{4a} \right) $$

或等价形式：

$$ \left( -\frac{b}{2a}, \ c - \frac{b^2}{4a} \right) $$

推导过程（配方法）：

$$ \begin{aligned} y &= ax^2 + bx + c \\ &= a\left(x^2 + \frac{b}{a}x\right) + c \\ &= a\left[x^2 + \frac{b}{a}x + \left(\frac{b}{2a}\right)^2 - \left(\frac{b}{2a}\right)^2\right] + c \\ &= a\left[\left(x + \frac{b}{2a}\right)^2 - \frac{b^2}{4a^2}\right] + c \\ &= a\left(x + \frac{b}{2a}\right)^2 + \left(c - \frac{b^2}{4a}\right) \end{aligned} $$

2.4 最值问题

  • 当

    $$a > 0$$

    时，函数开口向上，在顶点处取得最小值

    $$ y_{\text{min}} = \frac{4ac - b^2}{4a} = c - \frac{b^2}{4a} $$

    此时

    $$x = -\frac{b}{2a}$$

  • 当

    $$a < 0$$

    时，函数开口向下，在顶点处取得最大值

    $$ y_{\text{max}} = \frac{4ac - b^2}{4a} = c - \frac{b^2}{4a} $$

    此时

    $$x = -\frac{b}{2a}$$

2.5 对称轴

二次函数的图像是抛物线，其对称轴为：

$$ x = -\frac{b}{2a} $$

📊 第三部分：总结对比

概念 公式/性质 关联性
方程根 $$x = \frac{-b \pm \sqrt{\Delta}}{2a}$$ 函数与 x 轴交点横坐标
函数顶点 $$\left( -\frac{b}{2a}, \frac{4ac-b^2}{4a} \right)$$ 最值点、对称轴经过点
对称轴 $$x = -\frac{b}{2a}$$ 平分零点（若存在）
最值 $$y_{\text{极值}} = \frac{4ac-b^2}{4a}$$ 由 a 符号决定最大/最小

💡 记忆技巧与注意事项

  1. 符号记忆：

    • 顶点横坐标公式： $$x_v = -\frac{b}{2a}$$（负 b 除以 2a）
    • 纵坐标可用 $$y_v = f(x_v)$$计算，避免公式混淆

  2. 快速判断：

    • 开口方向：看 a 的符号
    • 与 y 轴交点：(0, c)
    • 对称轴位置：由 a, b 符号共同决定

  3. 验证方法：

    • 将求得的顶点代回原函数验证
    • 检查对称性：若 $$x_1, x_2$$是零点，则对称轴 $$x = \frac{x_1+x_2}{2}$$
